2.2.3. Regional Event Applications¶

The regional event application selects events for each building site. It writes event information in its corresponding building file (#-BIM.json), which was created by the Building Application, in the results folder.

The following options for regional event applications vary in the algorithm or method used to make the event assignment.

2.2.3.1. Nearest Neighbor Events (NNE)¶

Assigns events to building site by sampling from nearest neighboring event sites, using coordinate locations.
